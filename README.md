## CivicConnect

CivicConnect is a **smart city web application** designed to bridge the gap between citizens and municipal authorities.
It provides a simple and efficient way for people to report issues related to public infrastructure and track them easily.

## Project Overview

Many citizens face daily problems such as:

* Garbage not being collected
* Potholes on roads
* Water leakage
* Street light failures
* Stray animal issues

However, reporting these problems to the concerned authorities is often difficult.

**CivicConnect solves this problem** by providing a centralized platform where users can:

* Report complaints بسهولة (easily)
* Provide detailed information
* Help authorities take faster action

## Features :

Complaint Form

* User-friendly complaint submission form
* Validations for all input fields
* Includes:

  * Name, Phone, Email
  * Location & Landmark
  * Date & Time
  * Priority Level
  * Description

Top 5 Major Problems Section

Quick access complaint cards for common issues:

* Garbage Collection
* Pothole Repair
* Water Leakage
* Street Light Outage
* Stray Dogs

Clicking a card auto-opens the complaint form with pre-filled issue type.

FAQ Section

* Accordion-based FAQ design
* Helps users understand:

  * What CivicConnect is
  * Who can use it
  * How complaints work

About Page

* Explains:
  * Project purpose
  * Mission & Vision
  * Key features of the platform
